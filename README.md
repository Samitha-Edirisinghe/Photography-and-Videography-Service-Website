# Menaric - Premium Photography & Videography Website

A luxurious, highly animated photography and videography service website with a strong cinematic feel, built with React, TypeScript, and Framer Motion. Features extensive premium animations, parallax scrolling, and a sophisticated design system inspired by Apple, Aesop, and luxury fashion brands.


## ✨ Features

### 🎬 Premium Animations
- **Hero Slideshow**: 20+ layers of simultaneous animations with advanced 3D transitions
- **Parallax Scrolling**: Smooth multi-layer parallax effects throughout
- **Staggered Text Reveals**: Character-by-character title animations
- **Floating Particles**: Dynamic particle effects on backgrounds
- **Hover Effects**: Gold glow effects and micro-interactions
- **Smooth Page Transitions**: Seamless navigation between pages

### 📸 Core Functionality
- **Full-Screen Lightbox**: Professional portfolio gallery with keyboard navigation
- **Dynamic Masonry Grid**: Responsive image layouts
- **Auto-Rotating Testimonials**: Client testimonials with smooth transitions
- **Advanced Search**: Real-time search functionality with animated results
- **Newsletter Subscription**: Email signup with validation
- **Contact Form**: Multi-step contact form with validation
- **Dark/Light Mode**: Toggle between themes (light mode primary)

### 📱 Fully Responsive
- Mobile-first design approach
- Tablet and desktop optimized layouts
- Touch-friendly interactions
- Adaptive animations for performance

## 🛠️ Tech Stack

- **Framework**: React 18 with TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion (Motion)
- **Routing**: React Router v7 (Data Mode)
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Package Manager**: pnpm

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd menaric-wedding-films
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start development server**
   ```bash
   pnpm dev
   ```

4. **Build for production**
   ```bash
   pnpm build
   ```

5. **Preview production build**
   ```bash
   pnpm preview
   ```

## 🎯 Performance Optimization

- **Lazy Loading**: Images load on demand
- **Code Splitting**: Route-based code splitting
- **Motion Reduce**: Respects user preferences for reduced motion
- **Optimized Animations**: GPU-accelerated transforms
- **Image Optimization**: WebP format with fallbacks
